　　欢迎来到我的私人博客BeanBag。为了将来不忘记自己的前半生，我动笔写了一些自娱自乐的文字。

　　希望每位访客都能带着轻松慵懒的心情浏览这里的内容，感谢愿意在这些自言自语上驻留的你。

技术栈 🛠️

  • 框架：基于 Node.js 的 Hexo 框架
  • 部署节点：
    • GitHub Pages (国际访问)
    • Netlify (国内推荐)
  • 所用插件：
    • 本地搜索：Hexo-generator-searchdb
    • RSS订阅：Hexo-generator-feed
    • 夜间模式切换：Hexo-next-darkmode
    • 数学公式渲染：Hexo-filter-mathjax
    • 页面渲染优化：Hexo-optimize
    • 里世界访问限制：Hexo-hide-posts
